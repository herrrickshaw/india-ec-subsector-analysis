The Environmental Clearance approval funnel

How 113,970 EC proposals move from filing to grant on PARIVESH 2.0 (2022–2026). A current-state snapshot: each proposal shows its latest status, so proposals counted as "granted" have already passed through the ToR and appraisal stages.

Source: PARIVESH 2.0 open API (advanceSearchData?majorClearanceType=1), full EC register. 56 raw PARIVESH status values consolidated into canonical process stages (data/ec_funnel.json).
